* Logarithmic - log n 
* Linear Logarithmic - n log n

* Quadratic - n^2
* Cubic - n^3

------------*****------------

* Best 1(constant)

* 1 < log(log n) < sqrt(log n) < (log n)^2 < 2^(log n) < n < log(n!) < n(log(n)) < n^2 < 2^n < 4^n < n! 2^(2^n)

    * Worst Case -> Big O Notation
    * Best Case -> Omega Notation
